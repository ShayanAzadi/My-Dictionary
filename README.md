# My-Dictionary – C# Windows Forms
This is a user-driven dictionary application with no initial entries. Users add all words, definitions, and extra information themselves. The app lets you browse, search, manage, categorize, and even test yourself on your own entries—perfect for learning and building your personal vocabulary from scratch.


---

## Features
- Start with an **empty dictionary** and add words manually.  
- **Browse and search** words efficiently.  
- **Manage and categorize** your entries.  
- **Quiz yourself** on your added words to improve memory.  
- User-friendly interface built with Windows Forms.
