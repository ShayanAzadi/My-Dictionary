# My-Dictionary – C# Windows Forms

**Description:**  
My-Dictionary is a fully user-driven dictionary application built with C# and Windows Forms. **The dictionary starts empty**, and users add all words, definitions, and additional information themselves. Once entries are added, the app allows you to **browse, search, manage, categorize**, and even **test yourself** with quizzes based on your own entries.

## Features
- Start with an **empty dictionary** and add words manually.  
- **Browse and search** words efficiently.  
- **Manage and categorize** your entries.  
- **Quiz yourself** on your added words to improve memory.  
- User-friendly interface built with Windows Forms.  

## How to Use
1. Add a new word with its definition and optional extra information.  
2. Browse or search your dictionary entries.  
3. Organize entries into categories if needed.  
4. Use the quiz feature to test your knowledge of your own dictionary.  

## Notes
- The app **does not come with any predefined words**; everything starts empty.  
- All data is stored locally (text files or settings, depending on your implementation).  
- Ideal for learning, vocabulary building, or personal terminology management.
